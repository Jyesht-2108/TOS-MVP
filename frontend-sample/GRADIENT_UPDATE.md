# Dashboard Gradient Update 🌙

## New Gradient Applied

All dashboard header cards now use a beautiful **Deep Navy/Indigo Gradient** similar to modern learning platforms.

### Gradient Colors:
- **Left:** `#0F1C2E` (Deep Navy)
- **Middle:** `#1a2744` (Navy Blue)
- **Right:** `#1E2A55` (Indigo Blue)

### CSS Class:
```css
bg-gradient-to-r from-[#0F1C2E] via-[#1a2744] to-[#1E2A55]
```
